* Description :unfold:
This module adds support for [[https://github.com/casey/just][just]], a command runner for project-specific tasks.
Requires a working installation of ~just~.

- Syntax highlighting ([[doom-package:just-mode]] or [[doom-package:just-ts-mode]])
- LSP support ([[doom-package:just-lsp]])
- Format on save

** Module flags
- +lsp ::
  Enable LSP support. Requires [[doom-module::tools lsp]] and
  [[https://github.com/terror/just-lsp][just-lsp]].
- +tree-sitter ::
  Leverages tree-sitter for better syntax highlighting and structural text
  editing. Requires Emacs 29.1+ and [[doom-module::tools tree-sitter]].

** Packages
- [[doom-package:just-mode]] (without ~+tree-sitter~)
- [[doom-package:just-ts-mode]] (with ~+tree-sitter~)

* Installation
[[id:01cffea4-3329-45e2-a892-95a384ab2338][Enable this module in your ~doom!~ block.]]

This module requires a working installation of [[https://github.com/casey/just][just]].

*** Cargo
#+begin_src sh
cargo install just
#+end_src

*** Nix
#+begin_src nix
environment.systemPackages = [ pkgs.just ];
#+end_src

** LSP
Users with [[doom-module:+lsp]] enabled will need [[https://github.com/terror/just-lsp][just-lsp]].

*** Cargo
#+begin_src sh
cargo install just-lsp
#+end_src

*** Nix
#+begin_src nix
environment.systemPackages = [ pkgs.just-lsp ];
#+end_src

** Formatter
When [[doom-module:+lsp]] is enabled, ~just-lsp~ handles formatting via ~just --fmt
--unstable~. Otherwise, [[doom-module::editor format]] will invoke ~just~ directly.

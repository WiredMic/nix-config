* Description :unfold:
This module adds support for [[https://openscad.org/][OpenSCAD]], a solid 3D CAD modelling tool.
Requires a working installation of ~openscad~.

- Syntax highlighting ([[doom-package:scad-mode]])
- LSP support ([[doom-package:openscad-lsp]])
- Format on save

* Installation
[[id:01cffea4-3329-45e2-a892-95a384ab2338][Enable this module in your ~doom!~ block.]]

This module requires a working installation of [[https://openscad.org/downloads.html][OpenSCAD]].

*** Nix
#+begin_src nix
environment.systemPackages = [ pkgs.openscad ];
#+end_src

** LSP
Users with [[doom-module:+lsp]] enabled will need [[https://github.com/Leathong/openscad-LSP][openscad-lsp]].

*** Cargo
#+begin_src sh
cargo install openscad-lsp
#+end_src

*** Nix
#+begin_src nix
environment.systemPackages = [ pkgs.openscad-lsp ];
#+end_src

** Formatter (without +lsp)
Without [[doom-module:+lsp]], [[doom-module::editor format]] will invoke
[[https://github.com/hugheaves/scadformat][scadformat]] directly.

*** Pre-built binary
Download from the [[https://github.com/hugheaves/scadformat/releases][releases page]]
and place ~scadformat~ somewhere in your ~PATH~.

*** Go
#+begin_src sh
go install github.com/hugheaves/scadformat@latest
#+end_src
